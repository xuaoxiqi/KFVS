# Kinetic Flux Vector Splitting (KFVS) Scheme

## Code List

### KFVS1D

* KFVS1D.f90

* myE1PREX.F: exact 1D Riemann solver, developed by E.F.Toro (minor revision).

### KFVS2D

* KFVS2D_circular.f90
* KFVS2D_partial.f90

### TEST

* test1: 1D Sod shock tube problem
